# Maw3idi API

Backend API for a service appointment booking platform.

## Features

- JWT Authentication
- Role-based Authorization
- Provider Approval Workflow
- Availability Management
- Appointment Booking
- Double Booking Prevention
- Zod Validation
- Rate Limiting
- Helmet Security
- MongoDB Atlas

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Zod

## Run Locally

```bash
npm install
npm run dev
```

Environment variables:

```env
PORT=
MONGO_URI=
JWT_SECRET=
```
